Survey 1.0

SISTEMA PARA REALIZAÇÃO DE PESQUISA DE SATISFAÇÃO DE CLIENTES

Funcionalidades

* Clique nas funcionalidades para saber mais detalhes
Sistema configurado para ser exibido em tablets fixado em guichês de atendimento.
Opção de monstagem das perguntas e respostas personatilizadas, dependendo da necessidade.
Ao finalizar um questionário, uma nova pesquisa é gerado automáticamente ficando disponível para o próximo cliente.
Pesquisas de perguntas e respostas com várias opções de filtro.
Cadastro das perguntas vinculadas com as respostas disponibilizando questionários personalizados.
Formulário de detalhes contendo todas as informações em relação ao questionário.
Pesquisas dos questionários com várias opções de filtro.
Criação de questionários com perguntas e respostas automáticas.
Tela de detalhes com informações detalhadas do questionário e responstas.
Opção de impressão de relatórios simplificado e detalhado.
Questionários vinculados aos atendentes, tendo informações detalhadas do atendimento do mesmo.

Tecnologias

Neste projeto, foram utilizadas as seguintes tecnologias:


• Java 8
• Spring Boot
• Microserviços REST
• Spring Data JPA
• Banco de Dados PostgreSQL
• HTML, CSS e JavaScript
• JQuery
• Bootstrap
• Servidor Apache Tomcat
