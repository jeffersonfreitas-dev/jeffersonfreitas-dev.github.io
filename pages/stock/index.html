Stock Manager

SISTEMA PARA GERENCIAMENTO DE ALMOXARIFADO E ESTOQUES

Funcionalidades

* Clique nas funcionalidades para saber mais detalhes
• Pesquisas de produtos com várias opções de filtro.
• Controle de quantidade minima para aviso de produtos abaixo da quantidade estipulada.
• Controle de estoque separado por empresa.
• Tela de detalhes com informações de cadastros, estoque e movimentações.
• Histórico do produto com todas as movimentações (compras e saídas de solicitações).
• Pesquisas de compras com várias opções de filtro.
• Opção de anexar a nota fiscal da compra para facilitar futuras consultas.
• Entrada automática no estoque do produto no ato do cadastro.
• Tela de detalhes com informações de cadastros, estoque e fornecedores.
• Pesquisas de solicitações com várias opções de filtro.
• Validação de estoque disponível para saída, evitando estoque negativo.
• Filtro de produtos na hora da solicitação.
• Baixa automática no estoque no ato da solicitação.
• Tela de detalhes com informações de cadastros, estoque e solicitações.
• Emissão de solicitações com filtro de empresa, setor, solicitante e outros.
• Relatório detalhado das solicitações.
• Comprovante de entrega de mercadoria.
• Tela de detalhes com informações de cadastros, estoque e movimentações.
• Histórico do produto com todas as movimentações (compras e saídas de solicitações).

Tecnologias

Neste projeto, foram utilizadas as seguintes tecnologias:


• Java 11
• Spring Boot
• Spring Security
• Spring Data JPA
• Banco de Dados MySQL
• Engine Thymeleaf
• HTML, CSS e JavaScript
• JQuery
• Bootstrap
• Servidor Apache Tomcat
